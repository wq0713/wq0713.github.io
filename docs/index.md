# 个人技术博客

这里是基于 Markdown 的静态博客，支持：

- 顶部主目录（Tab）导航
- 左侧子目录（Sidebar）导航
- 右侧内容阅读区
- 全站搜索
- 黑色背景主题（默认暗色）

## 写作方式

1. 把文章放到 `docs/` 下面的对应目录（例如 `docs/docker/xxx.md`）
2. 导航会按 `docs/` 的文件夹结构自动生成（无需维护 `nav`）
3. 本地预览：`mkdocs serve`

## 目录示例

- Docker：容器、镜像、Compose
- Java：JVM、Spring
- 工具：Git、Linux
